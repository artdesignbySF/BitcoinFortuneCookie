# 🥠 Bitcoin Fortune Cookie

> **Status: Work in progress — actively under development**

Send and receive fortune cookies powered by the Bitcoin Lightning Network.

## What is this?

Bitcoin Fortune Cookie lets you open a digital fortune cookie for just a few satoshis, or send one to a friend via Nostr. Powered by real Bitcoin payments over the Lightning Network.

## Features

- 🥠 Open a random fortune cookie (100 sats)
- 📬 Send a fortune cookie to a Nostr address (100 sats)
- ✍️ Send a custom fortune cookie (1,100 sats)
- ⚡ Instant Lightning Network payments
- 🔐 No accounts, no tracking, no cookies (the browser kind)
- 🐦 Nostr integration — fortunes published to the decentralized social web

## Tech Stack

- React 18 frontend
- Node.js / Express backend
- MongoDB for cookie storage
- LND for Lightning Network payments
- nostr-tools for Nostr publishing
